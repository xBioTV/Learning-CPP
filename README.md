# Structure
This will be a loose way that this repo will be structured. Use this to understand the layout of this repo.

## Notes
This contains all notes on my journey to learning CPP. It's essentially my "programming diary." Within you will see a few files and folders.

### Project Notes.md
This will be notes relating to the corresponding project, so if a project is named "Image Tagger" the Project Notes.md will be in relation to that.
Each project will have their own Project Notes.md

### Definitions.md
This is a file that will live within the Notes folder where I am reading and writing technical definitions that relate to CPP. References for the definitions are inside there.

### Repo Notes.md
This is a file used to document my thoughts on the repo as a whole, and any changes that happen. More for historical use than anything else.
